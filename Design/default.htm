Atlanta Lending Library - Design

Introduction

The Atlanta project started life as an example of how to use NHibernate and Spring.Net. It has evolved into sample application of how to combine best practice techniques and technologies to create an application.

The project currently uses the following technologies:

  • NAnt;
  • NUnit;
  • NHibernate;
  • Spring.Net;
  • Silverlight;
  • NCover;
  • Firebird (RDBMS).

This remaining sections of this document describe the Development Methodology, Architecture (patterns and implementation), and functional design of the Atlanta Lending Library.
